Telepítési útmutató:

Győződj meg róla, hogy ezek telepítve vannak:
Node.js (ajánlott: 18+)
npm (Node.js-sel jön)

Git:
Ellenőrzés:
bash
node -v
npm -v
git --version

Projekt letöltése:
git clone https://github.com/TSzabo1222/SubMate.git
cd SubMate

Backend beállítása:
bash
cd backend
npm install

Backend indítása:
node server.js
vagy:
npm start

Frontend (Angular) beállítása:
Új terminálban:
cd frontend/login-app
npm install

Angular CLI telepítése (ha még nincs):
npm install -g @angular/cli

Angular Material hozzáadása:
ng add @angular/material

Ellenőrzés:
ng version

Frontend indítása:
Elsődleges módszer:
ng serve -o

Alternatíva (ha az `ng` nem működik):
npx ng serve -o

Elérés
Frontend:
http://localhost:4200

Backend:
http://localhost:3000

Gyakori hibák:
"ng is not recognized"
Telepítsd:
npm install -g @angular/cli
"MODULE_NOT_FOUND"

Hiányzó csomag:
npm install

"destination path already exists"

A mappa már létezik:
-töröld vagy
-használj másik mappanevet

Rossz parancs esetén:

Ne ezt használd:
npx serve

Hanem:
ng serve

Ha mindent jól csináltál:
Backend fut
Angular app betölt a böngészőben 🎉
